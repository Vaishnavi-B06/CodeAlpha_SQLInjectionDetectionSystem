🛡 Layer 1 — Input Security

  • ✅ Real-time SQL injection pattern matching
  • ✅ 24+ attack patterns across 4 threat levels
  • ✅ HTML entity decoding (evasion prevention)
  • ✅ Input sanitization before display
  • ✅ Client-side blocking before any network call
  • ✅ Threat logging with timestamp and context

🔐 Layer 2 — Access Security

  • ✅ Appwrite session management
  • ✅ JWT-based authentication tokens
  • ✅ Role-based access control (Admin / User)
  • ✅ Admin-only routes protected server-side
  • ✅ Automatic session expiry
  • ✅ AES-256-GCM encryption for sensitive data

☁️ Cloud Architecture

  • ✅ Appwrite Cloud (free tier) backend
  • ✅ Netlify-ready static frontend
  • ✅ No server-side code required
  • ✅ No database credentials in frontend
  • ✅ Appwrite handles credential storage
  • ✅ Accessible from any modern browser

🔒 Data Protection

  • ✅ AES-256-GCM via Web Crypto API
  • ✅ PBKDF2 key derivation (100k iterations)
  • ✅ Unique salt + IV per encryption operation
  • ✅ Passwords hashed by Appwrite (bcrypt)
  • ✅ Sensitive fields encrypted before storage
  • ✅ Security logs store sanitized input only
